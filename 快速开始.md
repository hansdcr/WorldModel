## 🐛 常见问题

### 问题 1: matplotlib 中文显示为方框

**解决方案**: 代码已包含字体配置，如果仍有问题：

```python
# 在代码开头添加
import matplotlib
matplotlib.rcParams['font.sans-serif'] = ['Arial Unicode MS']  # macOS
matplotlib.rcParams['axes.unicode_minus'] = False
```

### 问题 2: 虚拟环境未激活

**症状**: 运行代码时提示找不到模块

**解决方案**:
```bash
# 检查提示符是否有 (venv)
# 如果没有，运行:
source venv/bin/activate
```

### 问题 3: 代码运行没有输出

**原因**: matplotlib 窗口可能在后台

**解决方案**: 检查 Dock 栏中是否有 Python 图标，点击即可看到图形窗口
